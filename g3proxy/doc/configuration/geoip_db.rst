GeoIP DB
========

.. versionadded:: 1.7.25

This will set the db path for all GeoIP databases.

country
-------

**optional**, **type**: :ref:`file path <conf_value_file_path>`

Set the db file path for the country database.

asn
---

**optional**, **type**: :ref:`file path <conf_value_file_path>`

Set the db file path for the asn database.

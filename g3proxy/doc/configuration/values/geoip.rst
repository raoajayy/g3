GeoIP

.. _conf_value_iso_country_code:

iso country code
================

**yaml value**: str

The string should be ISO 3166 Alpha2 or Alpha3 code string.

.. _conf_value_continent_code:

continent code
==============

**yaml value**: str

The string should be:

  - AF, for Africa
  - AN, for Antarctica
  - AS, for Asia
  - EU, for Europe
  - NA, for North America
  - OC, for Oceania
  - SA, for South America
